Administración de Microdiseño Curricular

Proceso de Configuración

1

Exportar Plantilla Base

Descarga la plantilla actual como un archivo editable de Word (.docx) con todas las tablas e identificadores.

2

Edición en Word

Abre el archivo Word en tu computador local. Modifica textos, reordena tablas o parametriza nuevos campos utilizando el formato:

• [Campo: id_campo]: Input de texto
• [Texto: id_campo]: Área de texto
• [Numero: id_campo]: Input numérico
• [Lista: id | Op1; Op2; ...]: Desplegable
3

Subir Nueva Plantilla

Arrastra el archivo Word editado para sincronizar los cambios de estructura en el sistema de inmediato.

Arrastra tu archivo aquí o haz clic para explorar (.docx)
4

Actualización Automática

El backend procesará el documento y la nueva plantilla se propagará instantáneamente a las vistas de estudiantes y profesores.
